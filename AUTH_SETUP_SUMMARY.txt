DATABASE INTEGRATION
================================================================================
The website connects to two existing Supabase tables:

1. profiles table
   - id: uuid (references auth.users)
   - full_name: text
   - email: text
   - phone: text
   - move_timeline: text
   - persona: text ('military' | 'relocation' | 'discover')
   - created_at: timestamptz

2. saved_items table
   - id: uuid
   - user_id: uuid (references auth.users)
   - item_type: text ('listing' | 'tool' | 'page')
   - item_id: text
   - title: text
   - subtitle: text
   - route: text
   - metadata: jsonb
   - created_at: timestamptz
